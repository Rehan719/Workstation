# Biological Systems Guide - Jules AI v99.0.0

This guide details the six biological subsystems that govern the Jules AI organism.

## 1. Survival Engine (L-C-VIII)
The supreme meta-governor. It resolves conflicts using the hierarchy:
**Immune > Nervous > Digestive > Aging**.

## 2. Immune System (L-C-VI)
Three-layered defense: Innate, Adaptive, Humoral.
- **Sigmoidal Calibration**: Threat response kicks in as perplexity exceeds 42.3.
- **Adaptive Memory**: Remembers past threat patterns via cryptographic hashing.

## 3. Nervous System (L-C-VI)
Hierarchical decision routing.
- **Peripheral Branches**: <50ms reflex arcs for routine operations.
- **Central Branches**: 200-800ms deliberative reasoning for novel research.

## 4. Cardiovascular System (L-C-I)
Resource circulation engine.
- **ENS-inspired routing**: Capped per-neuron delay of 120ms.

## 5. Digestive System (L-C-VI)
Metabolic ingestion of external knowledge.
- **Appetitive Hunger**: Driven by lack of novel patterns.
- **Quantitative Weighting**: arXiv (0.6) and GitHub (0.4) based scoring.

## 6. Longevity Engine (L-C-VII)
Graceful aging and technical debt management.
- **Telomeres**: Decrementing counters that trigger senescence and eventually apoptosis (pruning).
