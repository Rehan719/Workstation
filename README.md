## Quick Start
1. **Prerequisites**:
   Ensure you have [Poetry](https://python-poetry.org/) installed.
2. **Setup**:
   ```bash
   poetry install
   ./scripts/init-secrets.sh
   ```
3. **Run Workstation**:
   ```bash
   poetry run streamlit run src/dashboard/app.py
   ```
4. **Run Discovery Simulation**:
   ```bash
   poetry run python run_discovery.py
   ```
