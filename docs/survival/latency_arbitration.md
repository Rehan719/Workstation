# Survival Instinct & Latency Arbitration

## Subsystem Hierarchy
The Survival Instinct Engine (Article BT/v3) resolves conflicts via strict hierarchy:
**Immune > Nervous > Digestive > Aging**

## Latency Budgets
- **Immune Veto**: ≤8.3 ms.
- **Nervous Arbitration**: ≤11.7 ms.
- **Digestive Recalibration**: ≤42 ms.

## v70.0 Enhancement: Allostatic Load
In the Mastery Edition, the `LatencyArbiter` monitors the **Molecular Triad**. If the **Redox Potential** deviates by >18 mV, the arbiter enters **High-Vigilance Mode**, effectively zeroing out the digestive budget to prioritize systemic survival.
