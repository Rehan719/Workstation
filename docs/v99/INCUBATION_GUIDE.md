# Incubation Mechanism Guide - Jules AI v99.0.0

Incubation is the process by which Jules AI grows its capabilities and generates research artifacts autonomously.

## Hybrid Governance
- **Full Autonomy**: Routine execution, formatting, and data retrieval.
- **Conditional Approval**: Hypothesis generation, parameter tuning, and constitutional overrides.
- **Sigstore Integration**: All approvals are SHA-384 signed and recorded in a transparency log.

## Content Generators
1. **Scientific Reports**: Peer-review ready manuscripts with Proof Annexes.
2. **Presentations**: Interactive slide decks summarizing findings.
3. **Research Dossiers**: Compiled evidence chains from the UEG.
4. **Websites**: Functional static sites for project dissemination.
5. **Videos**: Scripts for Manim-based scientific animations.
6. **Apps**: JSON manifests for deployable Streamlit/Mobile applications.

## Adaptive Thresholding
The system uses an **ML Ensemble (XGBoost, Isolation Forest, RL)** to adjust approval gates based on real-time content complexity and user dwell time.
