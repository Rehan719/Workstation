# Context-Aware Meta-Cognitive RL Portfolio (Article CQ)

## The 70/30 Innovation Split
Jules AI maintains a baseline ratio of 70% capability enhancement vs. 30% exploratory innovation.

## Weighting Factors

### Task Complexity
- **Cognitive Load ≥4.2**: Shifts ratio towards enhancement (80/20).
- **Cognitive Load <2.0**: Shifts ratio towards innovation (60/40).

### Performance Degradation
- **Latency >12.7% increase**: Triggers innovation push to escape local optima.
- **Accuracy >9.3% drop**: Triggers refinement push.

### User Engagement
- **Dwell Time >8.2s**: Reinforces current ratio.
- **Correction Rate >17%**: Pushes towards innovation.

### Domain Novelty
- **Lexical Divergence ≥68%**: Requires more hypothesis generation (Innovation).

## Online Optimization
Factor weights are optimized via online reinforcement learning, stored in the **Unified Evidence Graph (UEG)**.
