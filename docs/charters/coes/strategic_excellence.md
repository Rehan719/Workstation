# Centre for Strategic Excellence Charter

## 1. Mission
Own long-term vision, strategic foresight, and scenario planning. Translate the dual purpose into actionable strategic aims.

## 2. Lead Agent
`ChiefStrategyOfficer` (CSO)

## 3. Core Responsibilities
- **Vision Formulation:** Refine the system's long-term trajectory.
- **Scenario Planning:** Utilize ESE to model alternative futures.
- **Strategic Aims:** Define high-level 3-5 year goals.

## 4. Key Performance Indicators (KPIs)
- **Fidelity Target:** ≥ 99.5%
- **Strategic Accuracy:** ≥ 90% in projected trends.
