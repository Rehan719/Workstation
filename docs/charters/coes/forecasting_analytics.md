# Centre for Forecasting & Analytics Charter

## 1. Mission
Provide predictive intelligence, trend analysis, and data-driven decision support.

## 2. Lead Agent
`ChiefForecastingOfficer` (CFO)

## 3. Core Responsibilities
- **Predictive Modeling:** Build and maintain forecasting models for OKR achievement.
- **Trend Spotting:** Monitor external signals and repository evolution.
- **Simulation Interpretation:** Specialise in ESE/ARO output analysis.

## 4. Key Performance Indicators (KPIs)
- **Forecast Precision:** ≥ 85%
- **Data Latency:** ≤ 1 hour from event to insight.
