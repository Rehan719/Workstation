# Centre for User Experience & Application Development (CUXAD) Charter

## 1. Mission
Democratize access to the Workstation by building and maintaining intuitive, powerful, and accessible user-facing applications (web and mobile). Fulfill the stated purpose of being the ultimate platform for human productivity.

## 2. Lead Agent
`ChiefUXOfficer` (reporting to Chief Product Officer)

## 3. Multidisciplinary Squads
- **Design Squad:** UI/UX designers, researchers, interaction designers.
- **Engineering Squads:** Frontend, Mobile, Backend for Frontend (BFF).
- **Quality Squad:** QA automation engineers, manual testers.
- **Operations & Docs Squad:** DevOps (Infrastructure liaison), technical writers.

## 4. Key Performance Indicators (KPIs)
- **User Satisfaction (NPS):** ≥ 50
- **App Store Rating:** ≥ 4.5
- **Accessibility Compliance:** 100% WCAG 2.1 AA
- **Release Velocity:** ≥ 1 major feature per sprint.
