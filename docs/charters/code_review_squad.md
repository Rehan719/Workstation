# Charter: Code Review Squad

## 1. Mission
To enforce the highest expert standards of code quality, ensuring the Jules AI codebase is resilient, secure, and perfectly aligned with its Dual Purpose.

## 2. Squad Roles
- **Lead Architect**: Oversees structural integrity and design patterns.
- **Security Auditor**: Conducts vulnerability scans and pen-testing.
- **Performance Analyst**: Identifies bottlenecks and resource waste.
- **Purpose Guardian**: Verifies alignment with Articles 336-340.

## 3. Scope
Every line of code within:
- `agentic_core/` (The Brain)
- `src/` (The Interfaces)
- `infra/` (The Skeleton)
- `scripts/` (The Nervous System)

## 4. Operational Cycle
Audit → Plan → Execute → Verify → Review → Certify.
