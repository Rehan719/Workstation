# Oracle Always Free Setup

## 🚀 Free Tier Services
- **ARM Ampere A1 Compute**: 4 OCPUs and 24 GB of RAM.
- **AMD Compute**: 2 Micro instances.
- **Block Volume**: 200 GB total storage.
- **Autonomous Database**: 2 instances (20 GB each).

## 📝 Setup Instructions
1. Sign up for [Oracle Cloud Always Free](https://www.oracle.com/cloud/free/).
2. Create an API Key in the User Settings.
3. Configure the OCI CLI locally.
4. Set OCI environment variables for the deployment script.
