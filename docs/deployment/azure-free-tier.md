# Azure Free Account Setup

## 🚀 Free Tier Services
- **App Service**: 10 free web, mobile, or API apps.
- **Azure Functions**: 1 million requests per month (Always free).
- **Cosmos DB**: 1000 request units per second and 25 GB storage (Always free).
- **Virtual Machines**: 750 hours of B1s burstable VMs (12 months).

## 📝 Setup Instructions
1. Create an [Azure Free Account](https://azure.microsoft.com/free).
2. Use the Azure CLI or Portal to create a Service Principal.
3. Obtain the Tenant ID, Client ID, and Client Secret.
4. Set 'AZURE_CREDENTIALS' as a JSON string in your environment.
