# API Reference - Jules AI v99.0.0

## `agentic_core.orchestration`
### `BiologicalOrchestrator`
- `execute_scientific_task(task: dict)`: Primary entry point for task execution.
- `ingest_new_research(source: str, data: dict)`: Ingests external information.

## `agentic_core.incubation`
### `AdaptiveIncubationEngine`
- `start_incubation(goal: str, constraints: dict)`: Starts a new incubation cycle.

## `agentic_core.synthesis`
### `GrandSynthesisEngine`
- `run_synthesis()`: Triggers the foundational DNA generation.

## `agentic_core.immunity`
### `ImmuneSystem`
- `evaluate_threat(sample: dict)`: Returns threat score [0-1].

## `agentic_core.homeostasis`
### `HomeostaticRegulator`
- `update_metrics(telemetry: dict)`: Updates the 10-biomarker state.
