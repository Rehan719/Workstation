# Tiered Biological Parameterization (Article CP)

## The Three Tiers of Control

### Tier 1: Foundationally Fixed (Immutable)
Core constants that define the organism's existence.
- **Reflex Arc Latency**: <50 ms.
- **Cryptographic Key Sizes**: Fixed.
- **Formal Proof Axioms**: Non-negotiable.
- **Enforcement**: Survival Instinct veto (sub-50ms).

### Tier 2: Environmentally Tunable (Autonomic)
Homeostatically regulated parameters (±5% baseline).
- **Synaptic Scaling ($\tau_H$)**: 12-25s (Baseline ≈18.6s).
- **Learning Rates**: Workload-dependent.
- **XAI Sensitivity**: Context-aware.
- **Peristaltic Delay**: ≤120 ms.

### Tier 3: User-Controlled (Conscious)
Exposed via the Command Center for expert tuning.
- All Tier 2 parameters within safe ranges.
- Personalization preferences.

## Allostatic Integration
In v70.0, the `TieredParameterManager` integrates with the **Molecular Triad**. Parameter shifts are modulated by the current **ATP/ADP ratio** and **HSP90 turnover rate**.
