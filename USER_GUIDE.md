# Jules AI v60.0 User Guide

## Introduction
Jules AI v60.0 is an autonomous organism. You don't just "run" it; you "collaborate" with it.

## Key Workflows

### 1. Collaborative Research
- Input a high-level goal (e.g., "Find connections between CRISPR and Quantum Computing").
- The system starts an **Incubation Cycle**.
- You monitor progress in the **Command Center Dashboard**.

### 2. Autonomous Discovery
- Set the system to "Growth Mode".
- The **Digestive System** will pull recent papers from arXiv.
- The **Neuro-Symbolic Engine** will propose new causal hypotheses.
- Results are stored in the **Unified Evidence Graph (UEG)**.

### 3. Formal Verification
- Request a proof for any claim.
- The **Formal Proof Agent** will generate machine-checkable proofs using Lean/Coq/Z3.
- The **5-Layer Verification Pipeline** ensures unassailable truth.

## Troubleshooting
- **Latency Breaches**: Check resource allocation in the Dashboard.
- **High Allostatic Load**: The system may slow down to recover. This is normal homeostatic behavior.
- **Constitutional Blocks**: The system will refuse any action that violates the 77 Articles.
